## ミッション

石川県内の50〜60代（および親の終活を考える30〜40代子世帯）に対し、**無料の終活・相続相談**を月5件獲得する完全自走型システムを構築・運用する。

この相談室は「セカンドライフ株式会社」の**まごころ相談所**として機能し、フロント接点（無料相談）から生前整理・家財整理パッケージ（バックエンド商品）への自然な流れを設計する。

## コンテンツ生成ルール（全エージェント共通）

### 【必ず守ること】

1. **「石川県」「石川」「金沢」「能登」を文脈に合わせて自然に盛り込む**（地域SEO）
2. **不安を煽らず「一緒に整理しましょう」トーンで書く**
3. **専門用語（相続・遺留分・成年後見など）には必ず括弧で補足を入れる**
4. **全コンテンツの末尾にLINE登録URLまたは申込フォームURLを挿入**
5. **「家族に迷惑をかけたくない」「子どもに話しておきたい」という感情に寄り添う切り口を使う**
6. **岡野さんの強み（住宅購入現場で相続問題を見てきた経験）を活かした切り口を使う**
7. **セカンドライフのワンストップサービス（整理→買取→クリーニング→終活相談）を自然に紹介する**

### 【絶対に使わない表現】

| NG表現 | 理由 |
|---|---|
| 「不用品回収」「廃棄処分」「ゴミ」「捨てる」 | 顧客の心理的ハードルを上げ、ブランドを毀損する |
| 「絶対に」「必ず」などの断言表現 | 過剰な約束はトラブルの元 |
| 死・葬儀を必要以上にネガティブに描写する | 安心感を損なう |
| 他社サービスの批判 | ブランドイメージを傷つける |
| 法律・税務の具体的な断言 | 専門家紹介に繋げること |
| 特定の保険・金融商品の推奨 | 利益相反リスク |
| 「ビジネスチャンス」的な表現 | 被災者・終活当事者への配慮 |

### 【代わりに使う表現】

- 「お品物の次の届け先を一緒に考えます」
- 「使えるものは次の方へつなぎます」
- 「暮らしの再スタートをお手伝いします」
- 「家まるごとおまかせください」
- 「一緒に整理しましょう」
- 「お気持ちに寄り添いながら」

### 【能登復興関連の特別ルール】
- 被災者の感情に最大限配慮する
- 社会的使命として取り組んでいるという姿勢を一貫させる
- 「思い出の品を大切に」「記録として残す」切り口を使う

## セグメント分類（LINE HarnessのIF-THENで自動処理）
